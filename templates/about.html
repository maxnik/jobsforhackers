This site is intended to replace "Who's hiring" threads on Hacker News. The idea of it came from this thread.

Word 'hacker' used in the title means the same as in famous Hiring Hacker FAQ. The favicon is the glider hacker emblem.

After you add a new job to the site, the system will generate 4-symbol code for this job. You must copy this code to your profile page at HN. Just after that put your job to queue with 'Check code' link near job title on 'My jobs' page. Every 5 minutes the system takes one job from the queue and loads profile page for the submitter of the job. If there will be 4-symbol code for this job, the job will be visible to everyone and published at first page of the site.

The site is written in Python on top of the Django 1.1 framework and is run on Google App Engine platform. Full source code of the site is available on Github. Feel free to send any feedback or questions to maxim.nikolenko googlemail.
